# Support

Use the right channel for the fastest response.

## Bugs

Report reproducible SDK or documentation bugs with the GitHub bug report issue
template:

https://github.com/avery-so/alpha/issues/new?template=bug_report.md

Include the affected `@averyso/alpha` version, Node.js version, operating
system, minimal reproduction, expected behavior, and actual behavior.

## Feature Requests

Suggest product or SDK enhancements with the GitHub feature request issue
template:

https://github.com/avery-so/alpha/issues/new?template=feature_request.md

Describe the problem, proposed API or behavior, alternatives considered, and
any relevant context.

## Security Issues

Do not report vulnerabilities or leaked secrets in public issues. Follow
`SECURITY.md`, email `sec@avery.so`, or open a GitHub private security advisory.

## Usage Questions

Use GitHub Issues for general usage questions, integration questions, and
documentation gaps:

https://github.com/avery-so/alpha/issues/new?template=question.md

## Commercial and Integration Support

For commercial support, integration help, or partnership discussions, contact
`enterprise@avery.so`.
